# IRKTOURS.AI — предварительные маршруты

## Как запустить
1. Установи Node.js 18+.
2. Распакуй архив.
3. Выполни `npm install`.
4. Скопируй `.env.example` в `.env` и вставь `GIGACHAT_AUTH_KEY`.
5. Запусти `npm start`.
6. Открой `http://localhost:3000`.

## О проекте
Сайт делает не финальный тур, а **предварительный маршрут** для первого знакомства с поездкой. Дальше его можно уточнить у турагента или туроператора.

## Дополнительно
В форме добавлены параметры:
- сезон
- тип поездки
- темп маршрута
- выезд за город

Это помогает уменьшить “фантазирование” модели и делает ответ более аккуратным.
